# Mini-ATS – System Overview

## Purpose

This project is a minimal but production-ready Applicant Tracking System (ATS), designed to be shipped quickly to a first customer. The system supports multi-tenant usage, role-based access control, and a kanban-style candidate pipeline.

The main goal is to demonstrate:

- Rapid product development
- Clean system design
- Secure multi-tenant data handling
- Practical use of AI-assisted development

## Core Users

- **Admin** – internal users who manage customers and can act on their behalf
- **Customer** – companies using the ATS to manage jobs and candidates

## Key Features

- Authentication & authorization
- Job management
- Candidate management
- Kanban-based recruitment pipeline
- Admin-level user and organization management
- Configurable kanban-based recruitment pipeline
